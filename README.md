# Franco-grobler Tap

## How do I install these formulae?

`brew install franco-grobler/tap/<formula>`

Or `brew tap franco-grobler/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "franco-grobler/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
